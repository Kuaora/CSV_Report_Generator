# CSV_Report_Generator
Вывод данных из CSV файла в виде таблицы

`main.py` — консольная утилита для анализа и выборочного представления данных из CSV-файлов.

## Возможности
- Загрузка одного или нескольких CSV-файлов.
- Формирование отчёта по выбранным столбцам (`--report`).
- Удаление дублирующихся заголовков при объединении файлов.
- Подсчёт строк, содержащих указанную подстроку (`--count`).
- Вывод результата в табличном формате.

## Быстрый пример
```bash
python main.py --files employees1.csv employees2.csv --report performance team --count Python
